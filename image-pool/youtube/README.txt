# YouTube Image Pool (16:9 Landscape)

Add your 16:9 landscape images here for use in YouTube/horizontal format videos.

## Recommended Specifications:
- Aspect Ratio: 16:9
- Resolution: 1920x1080 or higher
- Format: JPG or PNG
- Quality: High resolution, well-lit, clear subject

## Example Filenames:
- nature-forest-001.jpg
- sunset-beach-001.jpg
- mountains-landscape-001.jpg
- city-skyline-001.jpg

## How to Use:
1. Add 20+ images to this folder
2. In your project config.json, set:
   {
     "aspectRatio": "16:9",
     "randomImages": true,
     "imageCount": 8,
     "useSharedPool": true
   }
3. Run batch processing - system will pick random images automatically!

## Tips:
- More images = more variety across videos
- Use descriptive filenames for easy management
- Keep consistent style/theme within pool
- Minimum 20 images recommended, 50+ ideal
