# TikTok Image Pool (9:16 Vertical)

Add your 9:16 portrait/vertical images here for use in TikTok/Shorts videos.

## Recommended Specifications:
- Aspect Ratio: 9:16
- Resolution: 1080x1920 or higher
- Format: JPG or PNG
- Quality: High resolution, mobile-optimized

## Example Filenames:
- portrait-person-001.jpg
- vertical-landscape-001.jpg
- phone-view-city-001.jpg

## How to Use:
1. Add 20+ vertical images to this folder
2. In your project config.json, set:
   {
     "aspectRatio": "9:16",
     "randomImages": true,
     "imageCount": 5,
     "useSharedPool": true
   }
3. Run batch processing!

## Tips:
- Use portrait-oriented images
- Consider mobile viewing experience
- Aim for 50-100 images for TikTok production
